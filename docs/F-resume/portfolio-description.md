# Portfolio Description

ThreatScope is a production-leaning cybersecurity portfolio project focused on Threat Intelligence, IOC Correlation, Detection Engineering, and SOC Operations. It is designed to demonstrate real-world security engineering skills for SOC Analyst, Threat Intelligence Analyst, Threat Hunter, and Detection Engineer interviews.

## Core Capabilities Demonstrated
- **Threat Intelligence**: Normalized ingestion, enrichment workflow, provenance tracking, confidence scoring
- **IOC Correlation**: Duplicate detection, cross-source correlation, relationship modeling
- **MITRE ATT&CK**: Technique mapping, coverage analytics, gap identification
- **Detection Engineering**: Sigma/YARA rule ingestion, validation, lifecycle management
- **SOC Operations**: Risk scoring, prioritization, report generation, alert triage support
- **Software Engineering**: FastAPI backend, React frontend, Docker, CI/CD, testing, documentation

## Architecture Highlights
- Async-ready FastAPI backend with service and router separation
- PostgreSQL for structured IOC/technique/rule storage
- Elasticsearch for fast IOC search and aggregation
- Docker Compose with healthchecks for reliable local runtime
- GitHub Actions CI with lint, tests, build validation, and security scanning
