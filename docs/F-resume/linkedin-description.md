# ThreatScope — LinkedIn Project Description

ThreatScope is a Cyber Threat Intelligence & IOC Correlation Platform I built to strengthen my portfolio for SOC Analyst, Threat Intelligence Analyst, and Detection Engineer roles.

## What it does
- Ingests, normalizes, and deduplicates threat indicators (IOCs)
- Correlates related IOCs and scores risk
- Maps findings to MITRE ATT&CK techniques
- Generates IOC summaries and technique coverage reports
- Supports Sigma and YARA rule management
- Exposes a FastAPI backend with a React frontend scaffold

## Tech Stack
Python, FastAPI, SQLAlchemy, PostgreSQL, Elasticsearch, React, TypeScript, Vite, Docker, GitHub Actions.

## Why I built it
I wanted a hands-on project that mirrors real SOC/CTI workflows and lets me discuss architecture, threat modeling, detection engineering, and incident response in interviews with confidence.
